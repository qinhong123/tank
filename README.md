# typescript、canvas来开发坦克游戏。

#### 1、目录结构

src/ 项目主文件

/src/canvas 渲染画布文件

/src/enum 定义的Ts枚举类型

/src/model 画布的模型实现方法

/src/service 图片，定位，音频的实现

/src/static 项目所需的图片

/src/util.ts 工具类

/src/config.ts 配置文件

/src/app.ts 主文件

/src/main.ts 入口文件

#### 2、pnpm管理工具

项目使用pnpm包管理工具

项目运行：pnpm dev

项目编译 :  pnpm run build
详细请看 package.json 文件
